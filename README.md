# Science Instruments Quiz App

This is a simple React quiz app built with Vite. It presents multiple-choice questions about scientific instruments and their uses.

## Features
- 5 questions about scientific instruments
- Multiple choice options
- Score display at the end
- Option to restart the quiz

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open your browser at [http://localhost:5173](http://localhost:5173)

## File Structure
- `src/questions.js`: Quiz questions
- `src/App.jsx`: Main quiz logic
- `src/App.css`: Styles
- `src/main.jsx`: Entry point
